Import Photos from a Gallery


Help


The Gallery Import application will allow you, the end-user, to import pictures from any Gallery (version 2 or 3) installation into your Facebook profile, as Photos Albums.

The first step you need to do in order to be able to import pictures into Facebook from a Gallery installation is install a small helper PHP file in your Gallery 2 installation, or a module on your Gallery 3 installation.

Gallery 2

  1. Create a new 'fb' subdirectory in your Gallery installation (using FTP or SSH), i.e. in the directory that contains main.php etc.
  2. Download the export_fb.phps helper file.
  3. Rename this file 'export_fb.php' and upload it in the new directory you just created ('fb').
The resulting script should be available at: http://your_gallery_url/fb/export_fb.php

Gallery 3

  1. Install the export_facebook module.

Note: You will not need to repeat this step each time you want to import pictures from Gallery to Facebook; this was a one time setup step.

Next, you can start an import session by clicking Gallery Import Applications bookmark, or if you just added the Gallery Import application, you've already been redirected to a new import session.

Once the import is completed, you'll be redirected to the last photos album that was created to import your pictures. You'll need to Accept all the photos that have been uploaded by the Gallery Import application before they'll be visible to others.

If you imported a Gallery album with more than 200 pictures, more than one photos album will have been created in Facebook. Make sure you Accept the pictures in all the new albums, not just the last one.

If you find any problem or inconsistency in your use of the Gallery Import application, please report them to the developer(s) using the application's discussion board.
